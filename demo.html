js-testing-api demo page

github.com/lilyseropian/js-testing-api

This page can be used to live test the "checker" testing API provided in api.js. The Whitelist and Blacklist input boxes both take comma-separated lists or a single word. The Structure input box takes a valid JSON object or a single word. See the documentation in api.js for more details. If the code in the textarea fails any of the tests, error messages will be shown at the bottom of the page.
